Pet Formulas

EXP multiplier:
  level 0-39   = 2
  level 40-49  = 3
  level 50-59  = 4
  level 60-69  = 5
  level 70-79  = 6
  level 80-89  = 7
  level 90+    = 8

requiredExp(level) = multiplier * (level + 1)

statValue(level, baseValue, statType) =
  baseValue + ((baseValue / denominator) * level)

denominator:
  AttackDamage = 10
  all other pet stats = 25

passiveBonus(level, rarity) =
  level * rarityMultiplier

attack + bonus =
  AttackDamage statValue + passiveBonus

Skill Formulas

EXP multiplier:
  level 0-49   = 2
  level 50-59  = 3
  level 60-69  = 4
  level 70+    = 5

requiredExp(level) = multiplier * (level + 1)

rarityMultiplier:
  Common    = 1
  Rare      = 2
  Epic      = 5
  Legendary = 10
  Mystic    = 15

bonusDamage(level, rarity) =
  level * rarityMultiplier

base + bonus =
  baseAttack + bonusDamage

simpleChain =
  base + bonus x multipleCount x bounceCount

DPS =
  simpleChain / cooldown

Target X:
  0 and 1 = generic in-game Target label
  2       = Highest HP
  3       = Random enemy

Wing Data

Extracted template fields:
  WingNum
  WingName
  WingIcon

Runtime instance fields:
  id, wingNum, rarity, level, currentExp, stats

Known native formulas:
  RarityGrowth = [30, 60, 90, 130, 200]
  RarityMaxExps = [40, 60, 80, 100, 120]

  MaxExp() = RarityMaxExps[rarity] * level

  GetStatValueAtLevel(type, targetLevel) =
    matchedStat.value + RarityGrowth[rarity] * (targetLevel - 1)

The fixed templates do not include base stats. WingInstance.stats is loaded from the
wings service response, so actual wing stat rolls appear server/generated.

Mount Data

Extracted template fields:
  MountNum
  MountName
  MountIcon
  MountObj

Runtime instance fields:
  id, mountNum, rarity, level, currentExp, stats

Known native formulas:
  RarityGrowth = [2, 4, 6, 8, 10]
  RarityMaxExps = [50, 70, 90, 110, 130]

  MaxExp() = RarityMaxExps[rarity] * level

  GetStatValueAtLevel(type, targetLevel) =
    matchedStat.value + RarityGrowth[rarity] * (targetLevel - 1)

The fixed templates do not include base stats. MountInstance.stats is loaded from
the mounts service response, actual mount stat rolls are server/generated.

Known Limits

Skill Base + bonus uses the recovered Skill.BonusDamage method. Simple chain multiplies that value by extracted multiple and bounce counts. Skill.AttackForText and Skill.BackgroundPattern also touch behavior objects and PvP/game-state data, so those runtime parts are not included here. Rarity upgrade per-level cost and time arrays are loaded through GearConfigDTO/server config; the APK only exposes crystalPricePerMinute = 2 and MaxRarityLevel = 90.
